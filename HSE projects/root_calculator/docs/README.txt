Программный продукт по извлечению квадратного корня из числа.

Описание
Root Calculator-кроссплатформенное графическое приложение для вычисления квадратного корня, на языке Python с использованием библиотеки Tkinter.

Соответствие требованиям технического задания:графический интерфейс, обработка ошибок, поддержка нескольких языков и возможность дальнейшего развития.

Возможности программы:
1)Извлечение квадратного корня положительных чисел
2)Вычисление квадратного корня из нуля
3)Корректная обработка некорректного пользовательского ввода
4)Ошибкоустойчивость
5)Графический пользовательский интерфейс
6)Переключение языков во время работы программы без перезапуска
7)Возможность добавления новых языков без пересборки программы
8)Сборка приложения в виде исполняемого файла
9)Модульная архитектура

Поддерживаемые платформы
macOS (сборка в формате .app)
Linux (запуск из исходного кода)
Windows (возможна сборка .exe)

Структура проекта

root_calculator:
    main.py – точка входа в программу
    math_core.py – модуль математических вычислений
    errors.py – обработка ошибок
    app_language.py – система локализации

  ..languages – файлы переводов
    ru.json
    en.json

  ..docs: – документация
  ..root_calculator_release: – готовые сборки приложения

README.txt – описание проекта

Поддержка языков

Локализация реализована с использованием JSON-файлов, размещённых в папке languages.

Пример файла ru.json:

{
  "title": "Калькулятор корней",
  "button": "Посчитать",
  "lang_ru": "RU",
  "lang_en": "EN",
  "result": "Результат"
}

Для добавления нового языка необходимо:
1)Создать новый JSON-файл в каталоге languages
2)Добавить переводы строк интерфейса
3)Язык станет доступен во время выполнения программы без перезапуска и перекомпиляции

Запуск программы из исходного кода

Требования:
 Python версии 3.9 или выше
 Наличие библиотеки Tkinter

Запуск осуществляется командой:
python main.py

Сборка приложения

Для создания автономного приложения используется PyInstaller.

Пример сборки:
pyinstaller main.spec

Готовый исполняемый файл размещается в каталоге dist.
